Detailed Description

MAC-STATUS-Resp primitive enables RLC to acknowledge a MAC-STATUS-Ind. It is possible that RLC would use this primitive to indicate that it has nothing to send or that it is in a suspended state or to indicate the current buffer occupancy to MAC.
